# 로고 에셋 (placeholder)

실제 엠블럼(풀/단순화 마크)은 별도 작업으로 SVG 벡터 트레이싱 후 이 폴더에 배치합니다.

- `emblem_full.svg` — 풀 엠블럼 (원형 인장 + 지도핀 + "옛길" 글자, 차콜/골드). 스플래시·앱스토어용.
- `mark_simple.svg` — 단순화 마크 (원 + 이음새 + 골드핀). 헤더·탭바·앱아이콘용. 16px까지 또렷해야 함.
- `app_icon.png` — 앱 아이콘 (mark_simple 기반, 1024x1024).

현재는 `mark_simple.svg`에 자리만 잡아둔 placeholder가 들어있습니다.
